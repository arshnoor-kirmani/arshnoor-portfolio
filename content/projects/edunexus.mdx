- Built a full-stack institute management platform with Next.js App Router, React, TypeScript, and MongoDB, using a hybrid rendering strategy: SSG for public/SEO pages, SSR for secure admin data fetching, and CSR for session-dependent dashboards.
- Created a 3-tier RBAC system (Super Admin, Teacher, Student) for student enrollment, attendance tracking, grade management, and academic records, with a protected dashboard and scoped permissions for each role.
- Designed and implemented 15+ REST API endpoints with Next.js Route Handlers and dual-layer Zod schema validation (client and server-side); managed global state with Redux Toolkit and form logic with React Hook Form for consistent, type-safe data flows.
- Secured the platform with NextAuth.js Credentials Provider, bcrypt.js password hashing, CSRF protection, and Next.js middleware for route-level authentication across dashboard and API endpoints.
- Achieved Lighthouse scores of 95+ Performance, 95+ Accessibility, 100 Best Practices, and 100 SEO through dynamic imports, lazy-loaded dashboard modules, and MongoDB indexing on email and studentId; deployed on Vercel with automated CI/CD and PR preview environments.
