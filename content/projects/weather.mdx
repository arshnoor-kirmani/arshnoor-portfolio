- Built a real-time weather dashboard with React, TypeScript, and Tailwind CSS, integrating three OpenWeatherMap APIs (Current Weather, 7-Day Forecast, Geocoding) to show live conditions and extended forecasts across thousands of global locations.
- Added browser Geolocation API support for automatic location detection and Leaflet.js maps with location markers and temperature overlays, making weather easier to explore by place.
- Improved the UX with debounced city search to reduce redundant API calls, manual response caching, skeleton loaders, error fallback UI, and Celsius/Fahrenheit plus metric/imperial unit toggles across a fully responsive mobile-first interface.
- Achieved sub-2s load times with manual API response caching and debounced search, cutting redundant network requests by ~40% during active city lookup sessions.
- Built a centralized unit conversion system supporting **5+ measurement formats** (Celsius, Fahrenheit, km/h, m/s, hPa).
